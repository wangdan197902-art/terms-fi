Pedagoginen agenssi

term_id: pedagogical_agent

Category: application_paradigms

Definition

Pedagoginen agenssi on ohjelmistokomponentti, joka on usein hahmottunut virtuaalihahmona ja toimii opettajana tai tutorina koulutusympäristöissä. Nämä agenssit hyödyntävät luonnollisen kielen käsittelyä…

Summary

Tekoälyentiteetti, joka on suunniteltu edistämään oppimista tarjoamalla opetusta, palautetta ja ohjausta.

Key Concepts

  • Älykäs tutorointi
  • Virtuaali ihminen
  • Mukautuva oppiminen
  • Ihmisen ja tietokoneen vuorovaikutus

Use Cases

  • Verkkokurssien avustajat
  • Kielenoppimissovellukset
  • Yrityskoulutuksen simuloinnit

Related Terms

  • Intelligent Tutoring System (älykäs tutorointijärjestelmä)
  • Chatbot (keskustelubotti)
