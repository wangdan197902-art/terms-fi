Vektori

term_id: vector

Category: basic_concepts

Definition

Tekoälyssä vektori on perustavanlaatuinen tietorakenne, jota käytetään tiedon numeeriseen esittämiseen. Se koostuu järjestetystä numeroluettelosta (alkioista), jotka kartoittavat entiteetin piirteet

Summary

Matemaattinen olio, joka edustaa suuretta, jolla on sekä suuruus että suunta, ja jota käytetään usein tekoälyssä datapisteen koodaamiseen korkeadimensioisessa avaruudessa.

Key Concepts

  • Dimensio
  • Upotus (Embedding)
  • Suuruus
  • Suunta

Use Cases

  • Semanttinen haku tietokannoissa
  • Luonnollisen kielen käsittelyn upotukset
  • Kuvantunnistuksen piirteiden erottaminen

Code Example

1
2
3

import numpy as np
vec = np.array([1.0, 2.0, 3.0])
print(vec.shape)

Related Terms

  • Matriisi (lukujen neliömäinen tai suorakulmainen taulukko)
  • Upotus (Embedding)
  • Kosinimuotoisuus (mittari kahden vektorin samankaltaisuudelle)
