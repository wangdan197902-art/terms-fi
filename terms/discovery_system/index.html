Discovery System

term_id: discovery_system

Category: basic_concepts

Definition

Discovery System on laskennallinen kehikko, jonka tavoitteena on kiihdyttää tieteellisiä tai analyyttisiä läpimurtoja automatisoimalla valtavien datatilavuuksien tutkiminen. Toisin kuin perinteinen optimointi, joka keskittyy olemassa olevien parametrien säätöön, discovery system etsii aktiivisesti uusia, yllättäviä korrelaatioita ja ratkaisuehdotuksia.

Summary

Automaattinen kehikko tai algoritmisen prosessin kokonaisuus, joka on suunniteltu tunnistamaan, tutkimaan ja arvioimaan potentiaalisia ratkaisuja, kuvioita tai hypoteeseja laajassa hakutilassa.

Key Concepts

  • Automaattinen tutkiminen
  • Hypoteesien generaatio
  • Aktiivinen oppiminen
  • Hakutilan navigointi

Use Cases

  • Uusien molekyylirakenteiden tunnistaminen lääkekehityksessä
  • Novelien materiaalien löytäminen tietyillä ominaisuuksilla
  • Odottamattomien korrelaatioiden löytäminen suurista datamääristä

Related Terms

  • Automated Machine Learning (AutoML)
  • Scientific Discovery (tieteellinen löytäminen)
  • Bayesian Optimization (bayesiläinen optimointi)
  • Data Mining (datankaivuu)
