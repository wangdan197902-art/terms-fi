Mode Collapse

term_id: mode_collapse

Category: basic_concepts

Definition

GAN-ympäristössä mode collapse ilmenee, kun generaattori oppii hyödyntämään diskriminaattorin heikkouksia tuottamalla vain kapean valikoiman uskottavia näytteitä, jättäen huomiotta datan jakautumisen muut osat. Tämä johtaa siihen, että malli menettää kykynsä tuottaa monipuolista sisältöä.

Summary

Mode collapse on generatiivisten vastustavien verkkojen (GAN) epäonnistumismuoto, jossa generaattori tuottaa vain rajoitetun määrän erilaisia tuloksia.

  • GAN:n vakaus

  • Diskriminaattorin palaute

  • Latentin avaruuden kattavuuden analysointi
